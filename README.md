# NYC 311 Citizen Grievance NLP 
A Natural Language Processing project to analyze and classify NYC 311 service requests/complaints using machine learning. 

#  Dataset
 - **Source:** [Kaggle - NYC 311 Service Requests](https://www.kaggle.com/datasets/new-york-city/nyc-311-service-requests)

 - **Sample Size:** 10,000 rows 

- **File:** `data/raw/311_service_requests_sample.csv` 

# Download Dataset 

 kaggle datasets download -d new-york-city/nyc-311-service-requests -p data/raw/ --unzip ``` 

**Note:** Requires [Kaggle API](https://github.com/Kaggle/kaggle-api) setup. 
Place your `kaggle.json` credentials in `~/.kaggle/`
